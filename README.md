# Personal-Budget-Planner
Description: A comprehensive web application designed to help users manage their personal finances effectively. The application features a user-friendly interface that allows for tracking income, monitoring bills, categorizing daily expenses, and managing debts.

Key Features:

Income Tracking: Log and view history of added funds with a real-time total calculator.

Visual Analytics: An interactive "Budget Overview" section featuring a chart to visualize financial status.

Expense Management: Dedicated modules for Bills, General Expenses, and Debts with budget limits and "remaining amount" indicators.

CRUD Functionality: Users can easily add, view, and delete entries to keep their records up to date.
